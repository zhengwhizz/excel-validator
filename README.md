<h1 align="center"> excel-validator </h1>

<p align="center"> laravel validator for excel/wps files.</p>

## Installing

```shell
$ composer require zhengwhizz/excel-validator -vvv
```

## Usage

```
$request->validate(['file' => 'required|excel']);
```
